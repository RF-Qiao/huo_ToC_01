# Prototype

This folder contains the original high-fidelity clickable prototype.

It is intentionally kept intact during the first architecture pass so the app remains runnable while the MVP foundation is introduced in:

- `src/types`
- `src/data`
- `src/services`
- `src/app`

Future work should extract one product flow at a time from this prototype instead of continuing to grow `PrototypeApp.tsx`.
